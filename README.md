# ejercicioMUTANTES
Permite verificar si un humano es mutante basándose en su secuencia de ADN. La verificación se realiza mediante un algoritmo que busca secuencias de cuatro letras iguales de forma horizontal, vertical y diagonal. El resultado de la verificación se guarda en una base de datos H2 y se pueden obtener estadísticas de las verificaciones realizadas.
